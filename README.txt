Für VSCode Tinymist installieren für den Language Support

Bevor du das Projekt startest, stelle sicher, dass du Folgendes installiert hast:

- [Node.js (empfohlen: LTS-Version)](https://nodejs.org/)
- [Git](https://git-scm.com/)
- [Expo Go (für Android/iOS)](https://expo.dev/client) – zum Scannen des QR-Codes auf dem echten Gerät
Expo CLI global: `npm install -g expo-cli`
npm install

Zum starten der Anwendung:

npx expo start
